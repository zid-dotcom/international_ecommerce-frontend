# Raynott E-com (Split files)

This project was generated by ChatGPT and split into separate pages and components.

Files included:
- src/pages: Home.jsx, Products.jsx, Booking.jsx, MyBookings.jsx, Auth.jsx, Contact.jsx, About.jsx, CartPage.jsx
- src/components: Header.jsx, Footer.jsx, ProductCard.jsx, Testimonial.jsx
- src/contexts: AuthContext.jsx, CartContext.jsx, BookingContext.jsx
- src/shared.js (mock products)
- src/App.jsx, src/main.jsx, src/index.css

How to run:
1. npm install
2. npm run dev
3. Replace firebase credentials in src/contexts/AuthContext.jsx to enable Google sign-in & Firestore.
